# Job assignment

## Description

Job assignment built with React.js and Scss, Bootstrap for general grid.
Web page with products from API in two sizes: small screen and large screen

## Installation

To install

`npm install`

To run

`npm start`

## To build

`npm run build`

Then use the dist folder you get.

## Built With

- React.js
- Scss
- Bootstrap 4
